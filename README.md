# 🐍 Python Exercises

Repositório dedicado aos meus estudos de **Python**, contendo diversos exercícios práticos utilizados para reforçar lógica de programação, POO e conceitos fundamentais da linguagem.

## 📌 Conteúdos trabalhados
Ao longo dos exercícios, pratiquei:

✔️ Programação Orientada a Objetos (POO)  
✔️ Criação de classes e métodos  
✔️ Encapsulamento e atributos privados  
✔️ Métodos para validação e cálculos  
✔️ Manipulação de listas e dados  
✔️ Estruturas de repetição e condição  
✔️ Organização de código e boas práticas  

## 📚 Sobre os exercícios
O repositório reúne atividades variadas — desde lógica básica até problemas mais estruturados — incluindo:

- Cálculo de médias e aprovação de alunos  
- Conversores e validações  
- Manipulação de valores numéricos  
- Exercícios gerais de prática em Python  

## 🎯 Objetivo
Aprimorar minha base em Python, desenvolvendo raciocínio lógico, familiaridade com a linguagem e habilidades essenciais para projetos maiores.
